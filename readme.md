## 📖 使用说明
### 👉 大模型应用开发岗
重点学习：
1. 第四章（部署与推理）：掌握工程化落地、性能优化、服务化方案
2. 第五章（提示词工程）：掌握Prompt开发、优化、企业级落地
3. 第六章（Agent与RAG）：掌握应用开发、系统架构设计、生产落地实践
4. 第七章（落地与实践）：掌握完整的项目落地流程、常见问题解决方案
### 👉 大模型算法岗
重点学习：
1. 第一章（基础与原理）、第二章（模型架构）：扎实掌握底层理论
2. 第三章（训练与微调）：掌握预训练、微调、对齐全流程技术
3. 第四章（部署与推理）：掌握推理优化、量化、部署核心原理
4. 第五章+第六章的🔬算法岗专属内容：掌握核心算法原理、优化思路、前沿进展
### 👉 通用面试技巧
1. 每个章节的「面试高频考点」优先掌握，是高频考题范围
2. 技术演进史部分可以帮助回答技术对比、发展趋势类问题，体现技术广度
3. 生产落地部分的内容可以应对场景设计、问题排查类面试题，体现实战能力
